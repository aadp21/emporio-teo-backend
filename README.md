# Emporio Teo - Backend (FastAPI)

Proyecto web para reemplazar el sistema local (Tkinter + Google Sheets) por una API REST con base de datos PostgreSQL.
Incluye módulos de **inventario**, **ventas**, **gastos** y **reportes**.

## 1. Requisitos

- Python 3.10+ (recomendado)
- PostgreSQL (local o remoto)
- pip

## 2. Instalación

```bash
git clone https://github.com/tu-usuario/emporio-teo-backend.git
cd emporio-teo-backend
python -m venv .venv
source .venv/bin/activate  # en Windows: .venv\Scripts\activate
pip install -r requirements.txt
"# emporio-teo-backend"
